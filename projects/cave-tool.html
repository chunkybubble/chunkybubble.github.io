Procedural Cave Tool

Type: Procedural Tool  •  Role: Programmer, Tech Artist  •  Team size: 1  •  Tools: Houdini, Unreal Engine, Houdini Engine

Overview

A procedural tool that creates unique cave systems with a spline editor. Takes a basic spline and randomly generates branches and rooms based of it.

My Contributions

  • Created and programmed the Houdini Digital Asset (HDA).
  • Integrated the HDA into Unreal Engine using Houdini Engine for designers/artists to quickly create cave systems.
  • Populate the caves with a variety of player collectibles, plants, and wildlife.

Technical Details

  • HDA crafted using operators and VEX to give randomness to cave systems and create data for objects to be placed inside caves.
  • Tool includes a button to regenerate all cave meshes in Unreal Engine if HDA is modified.
  • HDA contains several user-editable parameters to change cave randomness.
  • Tool includes a vector point position editor if user does not want to use spline editor.

What I Learned

  • The importance of pre-planning before execution. There are many ways to create caves, but I had to ask myself what type of caves would benefit the game.
  • The value of teamwork. There were many times while developing this where I got stuck and confused. Whether it's an easy or complicated problem, having another pair of eyes on it helped me work through it.
